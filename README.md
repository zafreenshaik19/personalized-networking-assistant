# 🤝 Personalized Networking Assistant

## Overview

This project helps users generate conversation starters for networking events. It also analyzes event topics, stores conversation history, and collects user feedback.

## Features

- Generate conversation starters
- Analyze event topics
- Save conversation history
- Wikipedia fact checker
- Feedback collection
- FastAPI backend
- Streamlit frontend
- Automated tests using pytest

## Technologies Used

- Python
- Streamlit
- FastAPI
- Pytest
- JSON

## How to Run

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit app

```bash
streamlit run app.py
```

### Run the FastAPI server

```bash
uvicorn main:app --reload
```

### Run tests

```bash
pytest
```
